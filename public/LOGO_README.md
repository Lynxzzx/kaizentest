# Logo do Kaizen

## Instruções

Para adicionar a logo do Kaizen ao site:

1. Coloque o arquivo da logo em: `public/logo.png`
   - Formato: PNG (recomendado) ou SVG
   - Tamanho recomendado: mínimo 400x130px para boa qualidade
   - Fundo: transparente (PNG) ou adaptado ao tema do site

2. Se a logo for branca sobre fundo preto:
   - Você pode criar duas versões:
     - `logo.png` - versão clara para fundos escuros
     - `logo-dark.png` - versão escura para fundos claros
   - Ou usar CSS para inverter cores conforme necessário

3. A logo será exibida automaticamente:
   - Na barra de navegação (topo)
   - Na homepage (hero section)
   - No footer (rodapé)

## Formatos Suportados

- PNG (recomendado)
- SVG (melhor para escalabilidade)
- JPG (não recomendado devido a fundo não transparente)

## Tamanhos Utilizados

- **Navegação (Navbar)**: ~160x50px
- **Homepage (Hero)**: ~200x65px  
- **Footer**: ~120x40px

O componente Logo ajusta automaticamente o tamanho conforme necessário.
